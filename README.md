# Demo Portfolio Website

A responsive and interactive frontend portfolio template built to demonstrate modern web development skills and UI/UX design principles. 

**Live Demo:** [View Deployed Site Here](https://demo-portfolio-website-alpha.vercel.app/)

> **Note:** This is a demonstration project created to showcase frontend development capabilities. It is not my active personal portfolio, but rather a functional template built to highlight specific tools and design patterns.

## 🚀 Tech Stack

- **React.js:** Used for building a component-driven, dynamic user interface.
- **Tailwind CSS:** Utilized for rapid, utility-first styling and ensuring a fully responsive design across all devices.
- **Email.js:** Integrated to handle client-side form submissions, allowing users to send emails directly from the contact form without a backend server.

## ✨ Features

- **Modern & Clean UI:** A visually appealing layout designed to highlight projects and skills effectively.
- **Fully Responsive:** Optimized for desktops, tablets, and mobile devices using Tailwind's utility classes.
- **Functional Contact Form:** Real-time email delivery powered by Email.js.
- **Smooth Navigation:** Seamless scrolling and component rendering.
